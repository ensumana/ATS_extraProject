# ATS_extraProject
Basic resume categorization

This Document specifies what you need to add on the app.

1. Add the Job description
2. Add the skills you require in the candidate
3. Add the list of resumes you want to process, basically like uploading.

You will be able to categorize the resumes based on candidate name, email, phone number,
education, experience, skills, score, category and why selected.

you will also get a downloadable file option through which you can download.
